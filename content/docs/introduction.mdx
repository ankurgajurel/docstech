# Introduction

DocsTech is a documentation infrastructure platform built for engineering organizations that operate at scale. It provides a unified authoring layer, real-time collaboration, and automated publishing pipelines вҖ” so your documentation ships as reliably as your code.

## Why Engineering Teams Choose DocsTech

- **MDX-native authoring** вҖ” Write in Markdown with full React component support. Embed live code samples, interactive diagrams, and API explorers directly into your pages.
- **Git-backed source of truth** вҖ” Content lives in your repository. Every change goes through your existing review workflow вҖ” pull requests, CI checks, and branch previews included.
- **Sub-second builds** вҖ” Incremental static generation with edge caching. Pages are pre-rendered at build time and served from the nearest PoP. Cold starts under 50ms.
- **Multi-tenant theming** вҖ” Ship white-labeled documentation portals for each product line or customer segment. Full control over layout, typography, and color systems.
- **Role-based access** вҖ” Granular permissions at the organization, team, and document level. Integrates with your existing IdP via SAML 2.0 and OIDC.
- **Observability built in** вҖ” Track page views, search queries, and API reference usage. Understand what your users actually read.

## Architecture Overview

DocsTech runs as a Next.js application with a content layer that reads MDX files from your repository at build time. The content pipeline handles frontmatter extraction, cross-reference resolution, search index generation, and OpenAPI spec ingestion automatically.

## Quick Example

Create an MDX file in your content directory with frontmatter metadata:

```mdx
---
title: Authentication
description: Configure SSO, API keys, and service account credentials.
---

# Authentication

DocsTech supports multiple authentication strategies for different
integration patterns. Choose the approach that fits your architecture.
```

The build pipeline detects new files, adds them to the navigation tree, generates search entries, and deploys the updated site вҖ” no manual configuration required.
